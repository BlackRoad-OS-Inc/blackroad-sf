🚀 Quick Start

Get up and running with local Ollama execution: 

# 1. Install Ollama on your Pi (ARM64)
curl -fsSL https://ollama.ai/install.sh | sh

# 2. Pull agent models
ollama pull llama3   # Lucidia
ollama pull mistral  # Octavia

# 3. Install project deps
npm ci

# 4. Run tests
npm test
